ℹ️ About ResumeAI

Everything you need to know about how the tool works

🤖 What is ResumeAI?

ResumeAI is an AI-powered recruitment screening tool that automatically analyses candidate resumes against job descriptions. It replaces the manual, time-consuming process of reading each resume by instantly evaluating candidates across 10 key dimensions.

Built for recruiters, HR teams, and hiring managers who want fast, consistent, and unbiased first-pass screening — without spending hours on every application.

⚙️ How It Works

You upload a resume (PDF, DOCX, or TXT) and paste a job description. The system extracts text from the file, combines it with the job description, and sends it to a large language model for deep analysis.

The AI evaluates technical skills, experience, education, certifications, projects, and more — then returns a structured JSON response that is displayed as a clean, readable report with a final eligibility verdict.

📊 Scoring Breakdown

Every resume is scored out of 100 using a weighted rubric that mirrors industry ATS standards:

CriteriaWeightPriority
Technical Skills Match40%High
Relevant Experience25%High
Projects Assessment15%Medium
Education10%Medium
Certifications5%Low
Soft Skills5%Low

🏆 Verdict System

Based on the final score, the candidate receives one of three verdicts:

✅
Selected
Score ≥ 80
🟡
Consider
Score 60–79
❌
Reject
Score < 60

📁 Supported File Formats

PDF — Most common resume format. Text is extracted using pypdf. Works best with text-based PDFs (not scanned images).

DOCX — Microsoft Word format. Full paragraph extraction using python-docx.

TXT — Plain text resumes. Directly read with UTF-8 encoding.
